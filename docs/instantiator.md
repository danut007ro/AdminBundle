# Instantiator

The `Instantiator` service is used to create non-shared objects that are preconfigured with default configuration and can be further customized. It can create instances for:

- [adapters](adapters/README.md)
- [formatters](formatters.md)
- [table configurators](table/README.md)
- [columns](table/columns.md)
- [batch actions](table/batch_actions.md)
